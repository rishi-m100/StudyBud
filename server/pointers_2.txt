### Study Guide: Pointers in C++
---

### Introduction to Pointers
1. **Definition of a pointer:**
   - A pointer is a variable that stores the memory address of another variable.

### First Declaration of a Pointer
1. **Declaration:**
   - `int* ptr;`

2. **Decomposition of the Declaration:**
   - `int* ptr;`
     - `int*` indicates that `ptr` is a pointer to an integer object.
     - `ptr` is the pointer variable that will store the memory address of an integer object.

3. **Explanation:**
   - `ptr` will eventually store the memory address of an integer object.

### Important Points to Remember
1. **Pointer Syntax:**
   - Use an asterisk (`*`) to declare a pointer.
   - Pointers are used to store memory addresses.
